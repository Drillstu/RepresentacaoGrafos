# RepresentacaoGrafos

Descrição
--------

Projeto em C que implementa uma representação de grafos usando matriz de
adjacência. Inclui operações básicas como limpar o grafo, adicionar e remover
arestas, exibir a matriz, calcular o grau de cada vértice e listar as
adjacências.

Requisitos
---------

- Compilador C (`gcc`)
- Sistema operacional: Linux (testado)

Como compilar
-------------

Compile o código usando `gcc`:

```bash
gcc -o RepresentacaoGrafos main.c
```

Como executar
-------------

Execute o binário gerado:

```bash
./RepresentacaoGrafos
```

Funcionalidades
--------------

- Inicializa um grafo com até 100 vértices
- Limpa a matriz de adjacência
- Insere e remove arestas entre vértices
- Exibe a matriz de adjacência atual
- Calcula e exibe o grau de cada vértice
- Lista as adjacências de cada vértice

Estrutura do repositório
------------------------

- `main.c` — código-fonte principal com a implementação do grafo
- `README.md` — documentação do projeto
